


# omni-
解释:
all 全部

## 记忆救星:omnipresent (adj)  
=omni + present = all+ present   
=无所不在(出现在所有地方)

## 延伸单词:  
omniscient(adj)
=all + know =全知的

omnidirectional(adj)
=全向的


## 进阶单词:  
omnivorous(adj)
=all + swallow = 杂食性的(所有都吃)  

omnipotent(adj)   
=all + powerful =全能的
